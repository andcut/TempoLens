# Stockfish license and attribution

TempoLens uses Stockfish for analysis. Stockfish is licensed under the GNU GPL v3.

If you distribute Stockfish binaries with TempoLens:
- Include the GPL v3 license text (`docs/licenses/GPL-3.0.txt`).
- Provide access to the corresponding Stockfish source code (for example, link to the official source or the exact commit you bundled).
- Clearly attribute Stockfish in your About/Licenses section.

Official source: https://github.com/official-stockfish/Stockfish
